WhatsApp Chat Analyzer

A Streamlit-based web application to analyze WhatsApp chat exports. This app provides detailed statistics, visualizations, and insights from your WhatsApp chats, including text analysis, activity trends, emoji usage, and media sharing patterns.

Live App: https://whatsapp-chat-analysis-nmv9pd3baksxnucouevfru.streamlit.app/

Features:
1) Top Statistics: Total messages, total words, media shared, and links shared.
2)Timelines:Monthly timeline of messages, Daily timeline of messages
3)Activity Analysis:Most busy days and months,Weekly activity heatmap
4)User Analysis:Identify the most active users in group chats
5)Text Analysis:WordCloud for most frequent words,Most common words bar chart
6)Emoji Analysis:Frequency of emojis,Pie chart visualization
7)Run the Streamlit app: streamlit run app.py

Usage: 
1) Upload your exported WhatsApp chat .txt file.
2) Select a user or “Overall” for group-wide analysis. And individual analysis also possible. Group chats, individual chats possible.
3) Click “Show Analysis” to generate all statistics and visualizations.
